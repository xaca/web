Mapa de imagen

Esta es una de las formas clásicas de agregar interactividad por medio de una imagen en un documento html, para crear un mapa, se deben combinar tres etiquetas, img, map, area. La primera etiqueta define la imagen que se va a usar como referencia para crear el mapa, el mapa define un mapa sobre la imagen y se vínculan a través del atributo usemap en la etiqueta img. Finalmente area define zonas sobre la imagen expresada en coordenadas.

Mapa de imagen

Existen 4 tipos de areas

  • default: Define una región completa
  • rect: Define una región rectangular (2 puntos)
  • circle: Define una región circular (3 puntos)
  • poly: Define un poligono (n puntos)
